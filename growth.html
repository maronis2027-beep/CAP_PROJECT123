Growth

A piece of work that I think demonstrates my growth is the “Make your own website” project from earlier this year, compared to the one we just did. The earlier project represents growth because with the knowledge I have now, I could do so much more with it. This project was in the early days of this class, so it is very very simplified, only using two types of coding blocks, and only having 33 lines of code. Moving on from this project was difficult because new ideas and blocks were introduced, and I personally believe that the turtle movements and learning how to incorporate variables in my code were the hardest for me to learn. In this project specifically, I faced many obstacles because of the fact that I had barely any coding experience, so it was like I was on my own for the first time. My perseverance through this project, along with many other future (and more difficult) projects shows my ability to learn quickly, and because of that, I was able to succeed. The one we did most recently, however, was so much more complicated. We had to type out our code fully instead of using blocks, and we had to learn new concepts such as html and style css. My completion of this project shows how much I have grown in comparison to the other project that we did in the beginning of the year.
